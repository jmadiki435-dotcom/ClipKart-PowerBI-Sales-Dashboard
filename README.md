# 📊 ClipKart Sales Dashboard (Power BI)

## Overview

ClipKart Sales Dashboard is a Power BI project built using a fictional e-commerce dataset. The dashboard provides business insights into sales performance, product trends, city-wise revenue, and key performance indicators.

This project demonstrates data cleaning, modeling, DAX measures, and interactive dashboard creation using Microsoft Power BI.

## Key Performance Indicators (KPIs)

- Total Revenue
- Total Orders
- Total Quantity Sold

---

## Dashboard Features

### Revenue by Category
Visualizes revenue generated by different product categories.

### Top Revenue Products
Displays the products generating the highest revenue.

### Top Performing Cities
Ranks cities based on total sales revenue.

### Price vs Revenue
Shows the relationship between product pricing and generated revenue.
